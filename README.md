# TaskManager
This project integrates data structures and algorithms into an application that manager your task.
## Setting

```bash
# Clone repository
https://github.com/tuanh2607/TaskManager.git

```

## Run application
How to run program step by step
```bash
# step 1 : Clone repository
git clone https://github.com/tuanh2607/TaskManager.git
# step 3 : Built program
g++ app\main.cpp -o program
# step 4 : Run program
.\program data.txt
```
